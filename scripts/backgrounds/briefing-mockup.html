Briefing Readability — Mockup Matrix

Three wallpapers (bright fog, golden mid-tone, dark night) × five briefing treatments. Each cell renders the briefing-greeting + date + briefing-prose roughly where they sit on desktop Today, using the same fonts. Copy is intentionally long (5 sentences, varied length) so multi-tone failures show up across multiple lines, not just the headline.

Baseline

What ships now — single soft shadow on white text. No scrim.

A · Top-edge scrim

Soft dark gradient in upper 50% of the page. Always-on, gentle.

B · Dual text-shadow

Tight crisp shadow + wide soft shadow. No scrim. Keeps "text on wallpaper" feel.

C · Backdrop-blur halo

Subtle frosted material hugging the text bounding box. Smooths multi-tone area.

A + B · Recommended

Gentle scrim + dual shadow. Reads on any wallpaper, presents as atmospheric design.
