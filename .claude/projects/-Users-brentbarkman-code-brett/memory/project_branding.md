Brett's logo is a faceted geometric crystal prism, warm amber/gold palette. Not a lettermark, not generic productivity iconography.

**Key decision:** The logo is a programmable SVG with CSS-driven light effects, NOT a static raster image. Midjourney output is reference art for shape/art direction only.

**Dynamic glow behavior:**
- Logo glows/pulses when chatting with Brett (AI is working/thinking)
- Glow warms up in morning, cools down at night (time-of-day awareness)
- Brighter/more saturated on busy days, calmer on light days
- Splash screen gets the full cinematic glow; nav version is quieter

**Why:** Amber/gold chosen over blue because blue is overused in productivity apps. Amber is distinctive, warm, matches Brett's personality (opinionated, warm, sharp), and is already the "today/now" accent color in the app.

**How to apply:** All logo implementations should use the SVG+CSS approach. The nav mark, splash screen, and any future branded surfaces should tap into CSS custom properties for glow color/intensity so they can be driven by app state.
