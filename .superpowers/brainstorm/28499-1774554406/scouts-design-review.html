Scouts Visual Prototype — Review

Two screens built in Pencil. Open the Pencil editor to see the full designs.

Screen 1: Scouts Roster Page

  • Full left nav with Scouts highlighted (purple badge)
  • 4 scout cards: 3 active + 1 completed (grayed out)
  • Each card: gradient avatar, name, status badge, goal summary, metadata (last run, findings count, cadence)
  • Elevated cadence shown in purple text on TSLA card
  • "New Scout" button in purple top-right

Screen 2: Scout Detail View (TSLA selected)

  • Collapsed nav + scout list (selected item has purple border)
  • Large avatar + name + active status + adaptive cadence status line
  • Edit / Pause action buttons
  • Config grid: Goal, Sources, Sensitivity, Cadence (showing base + elevated), Budget (38/60 runs)
  • Tabs: Findings (active, with count) | Activity Log
  • 3 finding types demonstrated:
    • Insight (purple icon) — "Unusual TSLA options volume"
    • Article (blue icon) — "BYD outsells Tesla in Q1"
    • Task (amber icon) — "Review TSLA position before earnings"
