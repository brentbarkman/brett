# Newsletter Email Integration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ingest email newsletters via Postmark Inbound Parse as content items in Inbox, with auto-discovery of new senders requiring user approval via in-app tasks.

**Architecture:** Postmark receives forwarded emails and POSTs to a webhook endpoint on the API. Known senders create content items directly; unknown senders create pending records + approval tasks. A Settings UI lets users manage senders and see the forwarding address.

**Tech Stack:** Hono (API routes), Prisma (DB), DOMPurify (HTML sanitization), Vitest (tests), React + Tailwind (Settings UI)

**Spec:** `docs/superpowers/specs/2026-04-07-newsletter-integration-design.md`

```typescript
import DOMPurify from "isomorphic-dompurify";
import { timingSafeEqual } from "crypto";
import { prisma } from "./prisma.js";
import { enqueueEmbed } from "@brett/ai";
import type { ContentMetadata } from "@brett/types";

// ── HTML Sanitization ──

const ALLOWED_CSS_PROPERTIES = new Set([
  "width", "max-width", "min-width",
  "height", "max-height", "min-height",
  "margin", "margin-top", "margin-right", "margin-bottom", "margin-left",
  "padding", "padding-top", "padding-right", "padding-bottom", "padding-left",
  "text-align", "vertical-align",
  "color", "background-color", "background",
  "font-size", "font-weight", "font-family", "font-style",
  "line-height", "letter-spacing",
  "border", "border-top", "border-right", "border-bottom", "border-left",
  "border-radius", "border-collapse", "border-spacing",
  "display", "table-layout",
]);

function sanitizeCssProperties(style: string): string {
  return style
    .split(";")
    .map((decl) => decl.trim())
    .filter((decl) => {
      if (!decl) return false;
      const colonIdx = decl.indexOf(":");
      if (colonIdx === -1) return false;
      const prop = decl.slice(0, colonIdx).trim().toLowerCase();
      const value = decl.slice(colonIdx + 1).trim().toLowerCase();
      // Block url() references (data exfiltration vector)
      if (value.includes("url(")) return false;
      // Block position: fixed/absolute (overlay attacks)
      if (prop === "position" && (value === "fixed" || value === "absolute")) return false;
      // Block high z-index
      if (prop === "z-index") return false;
      return ALLOWED_CSS_PROPERTIES.has(prop);
    })
    .join("; ");
}

/**
 * Sanitize newsletter HTML for safe rendering.
 * If htmlBody is empty/null, falls back to textBody wrapped in <pre>.
 */
export function sanitizeNewsletterHtml(htmlBody: string, textBody?: string | null): string {
  if (!htmlBody && textBody) {
    // Escape HTML entities in plain text, then wrap in <pre>
    const escaped = textBody
      .replace(/&/g, "&amp;")
      .replace(/</g, "&lt;")
      .replace(/>/g, "&gt;");
    return `<pre style="white-space:pre-wrap;word-wrap:break-word">${escaped}</pre>`;
  }

  if (!htmlBody) return "";

  // Step 1: DOMPurify pass
  const purified = DOMPurify.sanitize(htmlBody, {
    ALLOWED_TAGS: [
      "div", "span", "p", "br", "hr",
      "h1", "h2", "h3", "h4", "h5", "h6",
      "strong", "b", "em", "i", "u", "s", "sub", "sup",
      "ul", "ol", "li",
      "a", "img",
      "table", "thead", "tbody", "tr", "td", "th",
      "blockquote", "pre", "code",
    ],
    ALLOWED_ATTR: [
      "href", "src", "alt", "title", "width", "height",
      "style",
      "class", "id",
      "target", "rel",
      "colspan", "rowspan", "cellpadding", "cellspacing",
    ],
    FORBID_TAGS: ["script", "iframe", "object", "embed", "form", "input", "button", "textarea", "select", "style"],
    FORBID_ATTR: ["onerror", "onload", "onclick", "onmouseover", "onmouseout", "onfocus", "onblur"],
    ALLOW_DATA_ATTR: false,
  });

  // Step 2: Sanitize inline styles + force link safety
  // Parse as DOM fragment, walk nodes
  const doc = DOMPurify.sanitize(purified, { RETURN_DOM: true }) as unknown as DocumentFragment;
  const container = doc as unknown as HTMLElement;

  const walker = document.createTreeWalker(container, NodeFilter.SHOW_ELEMENT);
  let node: Node | null = walker.currentNode;
  while (node) {
    const el = node as HTMLElement;
    // Sanitize style attributes
    if (el.hasAttribute?.("style")) {
      const cleaned = sanitizeCssProperties(el.getAttribute("style")!);
      if (cleaned) {
        el.setAttribute("style", cleaned);
      } else {
        el.removeAttribute("style");
      }
    }
    // Force safe link behavior
    if (el.tagName === "A") {
      el.setAttribute("target", "_blank");
      el.setAttribute("rel", "noopener noreferrer");
    }
    node = walker.nextNode();
  }

  return container.innerHTML ?? "";
}

/**
 * Extract bare email from a From header that may include a display name.
 * Handles both "dan@example.com" and "Dan <dan@example.com>" formats.
 */
export function extractEmail(from: string): string {
  const match = from.match(/<([^>]+)>/);
  return (match ? match[1] : from).trim().toLowerCase();
}

// ── Secret Verification ──

export function verifyIngestSecret(provided: string, expected: string): boolean {
  if (provided.length !== expected.length) return false;
  return timingSafeEqual(Buffer.from(provided), Buffer.from(expected));
}

// ── Item Creation ──

export async function ingestNewsletter(params: {
  userId: string;
  senderName: string;
  senderEmail: string;
  subject: string;
  htmlBody: string;
  textBody?: string | null;
  messageId: string;
  receivedAt: string;
}): Promise<{ itemId: string }> {
  const sanitizedBody = sanitizeNewsletterHtml(params.htmlBody, params.textBody);

  const metadata: ContentMetadata = {
    type: "newsletter",
    senderName: params.senderName,
    senderEmail: params.senderEmail,
    issueSubject: params.subject,
    receivedAt: params.receivedAt,
  };

  const item = await prisma.item.create({
    data: {
      type: "content",
      status: "active",
      title: params.subject,
      contentType: "newsletter",
      contentStatus: "extracted", // already have the full content
      contentTitle: params.subject,
      contentBody: sanitizedBody,
      contentMetadata: metadata as any,
      source: params.senderName,
      sourceId: params.messageId,
      userId: params.userId,
    },
  });

  // Fire-and-forget embedding for search
  enqueueEmbed({ entityType: "item", entityId: item.id, userId: params.userId });

  return { itemId: item.id };
}

// ── Pending Newsletter + Approval Task ──

export async function createPendingNewsletter(params: {
  userId: string;
  senderName: string;
  senderEmail: string;
  subject: string;
  htmlBody: string;
  textBody?: string | null;
  messageId: string;
  receivedAt: string;
}): Promise<{ pendingId: string; taskId: string }> {
  // Check dedup on pending
  const existingPending = await prisma.pendingNewsletter.findFirst({
    where: { postmarkMessageId: params.messageId },
  });
  if (existingPending) {
    return { pendingId: existingPending.id, taskId: existingPending.approvalItemId ?? "" };
  }

  // Check if there's already an approval task for this sender (avoid spamming tasks)
  const existingTask = await prisma.item.findFirst({
    where: {
      userId: params.userId,
      source: "Brett",
      sourceId: { startsWith: `newsletter-approve:${params.senderEmail.toLowerCase()}` },
      status: "active",
    },
  });

  // Create pending record
  const pending = await prisma.pendingNewsletter.create({
    data: {
      userId: params.userId,
      senderEmail: params.senderEmail.toLowerCase(),
      senderName: params.senderName,
      subject: params.subject,
      htmlBody: params.htmlBody,
      textBody: params.textBody,
      postmarkMessageId: params.messageId,
      receivedAt: new Date(params.receivedAt),
      approvalItemId: existingTask?.id ?? null,
    },
  });

  // Only create a new approval task if one doesn't exist for this sender
  if (existingTask) {
    return { pendingId: pending.id, taskId: existingTask.id };
  }

  const task = await prisma.item.create({
    data: {
      type: "task",
      title: `Approve newsletter sender: ${params.senderName} (${params.senderEmail})`,
      source: "Brett",
      sourceId: `newsletter-approve:${params.senderEmail.toLowerCase()}`,
      status: "active",
      userId: params.userId,
      contentMetadata: {
        newsletterApproval: true,
        pendingNewsletterId: pending.id,
        senderEmail: params.senderEmail.toLowerCase(),
        senderName: params.senderName,
      },
    },
  });

  // Link back to the task
  await prisma.pendingNewsletter.update({
    where: { id: pending.id },
    data: { approvalItemId: task.id },
  });

  return { pendingId: pending.id, taskId: task.id };
}

// ── Approve / Block ──

const MAX_RETROACTIVE_INGEST = 10;

export async function approveSender(userId: string, pendingId: string): Promise<{ senderId: string; ingestedCount: number }> {
  const pending = await prisma.pendingNewsletter.findFirst({
    where: { id: pendingId, userId },
  });
  if (!pending) throw new Error("Pending newsletter not found");

  // Create or update the sender record
  const sender = await prisma.newsletterSender.upsert({
    where: { userId_email: { userId, email: pending.senderEmail } },
    create: {
      userId,
      name: pending.senderName,
      email: pending.senderEmail, // already lowercase
      active: true,
    },
    update: { active: true, name: pending.senderName },
  });

  // Retroactively ingest pending newsletters from this sender (most recent first, capped)
  const allPending = await prisma.pendingNewsletter.findMany({
    where: { userId, senderEmail: pending.senderEmail },
    orderBy: { receivedAt: "desc" },
    take: MAX_RETROACTIVE_INGEST,
  });

  let ingestedCount = 0;
  for (const p of allPending) {
    // Dedup: skip if already ingested
    const exists = await prisma.item.findFirst({
      where: { userId, sourceId: p.postmarkMessageId },
    });
    if (!exists) {
      await ingestNewsletter({
        userId,
        senderName: sender.name,
        senderEmail: sender.email,
        subject: p.subject,
        htmlBody: p.htmlBody,
        textBody: p.textBody,
        messageId: p.postmarkMessageId,
        receivedAt: p.receivedAt.toISOString(),
      });
      ingestedCount++;
    }
  }

  // Count total pending for logging
  const totalPending = await prisma.pendingNewsletter.count({
    where: { userId, senderEmail: pending.senderEmail },
  });
  if (totalPending > MAX_RETROACTIVE_INGEST) {
    console.warn(`[newsletter] ${totalPending - MAX_RETROACTIVE_INGEST} pending newsletters from ${pending.senderEmail} exceeded retroactive cap, discarded`);
  }

  // Clean up all pending records from this sender
  await prisma.pendingNewsletter.deleteMany({
    where: { userId, senderEmail: pending.senderEmail },
  });

  // Complete the approval task
  if (pending.approvalItemId) {
    await prisma.item.update({
      where: { id: pending.approvalItemId },
      data: { status: "done", completedAt: new Date() },
    });
  }

  return { senderId: sender.id, ingestedCount };
}

export async function blockSender(userId: string, pendingId: string): Promise<void> {
  const pending = await prisma.pendingNewsletter.findFirst({
    where: { id: pendingId, userId },
  });
  if (!pending) throw new Error("Pending newsletter not found");

  // Create blocked sender record
  await prisma.newsletterSender.upsert({
    where: { userId_email: { userId, email: pending.senderEmail } },
    create: {
      userId,
      name: pending.senderName,
      email: pending.senderEmail,
      active: false, // blocked
    },
    update: { active: false },
  });

  // Delete all pending from this sender
  await prisma.pendingNewsletter.deleteMany({
    where: { userId, senderEmail: pending.senderEmail },
  });

  // Complete the approval task
  if (pending.approvalItemId) {
    await prisma.item.update({
      where: { id: pending.approvalItemId },
      data: { status: "done", completedAt: new Date() },
    });
  }
}

// ── Cleanup ──

export async function cleanupExpiredPending(): Promise<number> {
  const cutoff = new Date(Date.now() - 30 * 24 * 60 * 60 * 1000);

  // Only clean up records whose approval task is already resolved or doesn't exist
  const expired = await prisma.pendingNewsletter.findMany({
    where: {
      createdAt: { lt: cutoff },
    },
  });

  let cleaned = 0;
  for (const p of expired) {
    // Check if approval task still active
    if (p.approvalItemId) {
      const task = await prisma.item.findUnique({
        where: { id: p.approvalItemId },
        select: { status: true },
      });
      // Skip if task is still active — user hasn't decided yet
      if (task && (task.status === "active" || task.status === "snoozed")) {
        continue;
      }
    }

    await prisma.pendingNewsletter.delete({ where: { id: p.id } });
    cleaned++;
  }

  return cleaned;
}

  ingestNewsletter,

// ── Webhook (no auth — secret-gated) ──

const webhookRouter = new Hono();

webhookRouter.post("/email/ingest/:secret", async (c) => {
  const secret = c.req.param("secret");
  const expectedSecret = process.env.NEWSLETTER_INGEST_SECRET;

  if (!expectedSecret || !verifyIngestSecret(secret, expectedSecret)) {
    return c.json({ error: "Unauthorized" }, 401);
  }

  let body: Record<string, unknown>;
  try {
    body = await c.req.json();
  } catch {
    return c.json({ ok: true }); // malformed → 200, no retry
  }

  const from = typeof body.From === "string" ? body.From : null;
  const fromName = typeof body.FromName === "string" ? body.FromName : from ?? "Unknown";
  const subject = typeof body.Subject === "string" ? body.Subject : null;
  const htmlBody = typeof body.HtmlBody === "string" ? body.HtmlBody : "";
  const textBody = typeof body.TextBody === "string" ? body.TextBody : null;
  const date = typeof body.Date === "string" ? body.Date : new Date().toISOString();
  const messageId = typeof body.MessageID === "string" ? body.MessageID : null;

  // Require at minimum: From, Subject, and either HtmlBody or TextBody
  if (!from || !subject || (!htmlBody && !textBody) || !messageId) {
    return c.json({ ok: true }); // malformed → 200
  }

  // Size guard — reject oversized payloads (2MB HTML max)
  const MAX_HTML_SIZE = 2 * 1024 * 1024;
  if (htmlBody.length > MAX_HTML_SIZE) {
    console.warn(`[newsletter] HTML body too large (${htmlBody.length} bytes), dropping`);
    return c.json({ ok: true });
  }

  // Resolve user (v1: single user from env)
  const userId = process.env.NEWSLETTER_INGEST_USER_ID;
  if (!userId) {
    console.error("[newsletter] NEWSLETTER_INGEST_USER_ID not set");
    return c.json({ error: "Server misconfigured" }, 500);
  }

  // Dedup by MessageID
  const existing = await prisma.item.findFirst({
    where: { userId, sourceId: messageId },
  });
  if (existing) return c.json({ ok: true }); // idempotent

  // Sender lookup (case-insensitive)
  const sender = await findApprovedSender(userId, from);

  if (sender) {
    if (!sender.active) {
      // Blocked sender — silently drop
      return c.json({ ok: true });
    }

    // Known active sender — ingest
    try {
      await ingestNewsletter({
        userId,
        senderName: sender.name,
        senderEmail: sender.email,
        subject,
        htmlBody,
        textBody,
        messageId,
        receivedAt: date,
      });
    } catch (err) {
      console.error("[newsletter] Ingest failed:", err);
      return c.json({ error: "Internal error" }, 500);
    }
  } else {
    // Unknown sender — create pending + approval task
    try {
      await createPendingNewsletter({
        userId,
        senderName: fromName,
        senderEmail: from,
        subject,
        htmlBody,
        textBody,
        messageId,
        receivedAt: date,
      });
    } catch (err) {
      console.error("[newsletter] Pending creation failed:", err);
      return c.json({ error: "Internal error" }, 500);
    }
  }

  return c.json({ ok: true });
});

senderRouter.patch("/:id", async (c) => {
  const user = c.get("user");
  const id = c.req.param("id");
  const body = await c.req.json();

  const sender = await prisma.newsletterSender.findFirst({
    where: { id, userId: user.id },
  });
  if (!sender) return c.json({ error: "Not found" }, 404);

  const data: Record<string, unknown> = {};
  if (typeof body.name === "string") data.name = body.name;
  if (typeof body.active === "boolean") data.active = body.active;

  const updated = await prisma.newsletterSender.update({
    where: { id },
    data,
  });
  return c.json(updated);
});

  });

  await prisma.newsletterSender.delete({ where: { id } });
  return c.json({ ok: true });

function NewsletterPreview({
  contentBody,
  contentMetadata,
}: {
  contentBody?: string;
  contentMetadata?: ContentMetadata;
}) {
  const senderName = contentMetadata?.type === "newsletter" ? contentMetadata.senderName : undefined;
  const receivedAt = contentMetadata?.type === "newsletter" ? contentMetadata.receivedAt : undefined;

  // Content is already sanitized server-side, but belt-and-suspenders:
  // render inside a sandboxed iframe for defense-in-depth.
  const iframeContent = useMemo(() => {
    if (!contentBody) return "";
    // Re-sanitize client-side as defense-in-depth
    const sanitized = newsletterPurify.sanitize(contentBody, {
      ALLOWED_TAGS: [
        "div", "span", "p", "br", "hr",
        "h1", "h2", "h3", "h4", "h5", "h6",
        "strong", "b", "em", "i", "u", "s", "sub", "sup",
        "ul", "ol", "li",
        "a", "img",
        "table", "thead", "tbody", "tr", "td", "th",
        "blockquote", "pre", "code",
      ],
      ALLOWED_ATTR: [
        "href", "src", "alt", "title", "width", "height",
        "style", "class", "id",
        "target", "rel",
        "colspan", "rowspan", "cellpadding", "cellspacing",
      ],
      FORBID_TAGS: ["script", "iframe", "object", "embed", "form", "input", "button", "textarea", "select", "style"],
      FORBID_ATTR: ["onerror", "onload", "onclick", "onmouseover"],
      ALLOW_DATA_ATTR: false,
    });

    // Wrap in a full HTML doc for the sandboxed iframe
    return `<!DOCTYPE html>
<html><head><style>
  body {
    font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
    font-size: 14px;
    line-height: 1.6;
    color: rgba(255,255,255,0.85);
    background: transparent;
    max-width: 680px;
    margin: 0 auto;
    padding: 0;
    overflow-x: hidden;
  }
  a { color: #D4AF37; }
  img { max-width: 100%; height: auto; }
  table { max-width: 100%; }
</style></head><body>${sanitized}</body></html>`;
  }, [contentBody]);

  return (
    <div className="space-y-2">
      {/* Metadata pill */}
      {(senderName || receivedAt) && (
        <div className="flex items-center gap-2">
          {senderName && (
            <span className="text-xs text-white/50 font-medium">{senderName}</span>
          )}
          {receivedAt && (
            <span className="text-xs text-white/30">
              {new Date(receivedAt).toLocaleDateString(undefined, { month: "short", day: "numeric", year: "numeric" })}
            </span>
          )}
        </div>
      )}

      {/* Sandboxed iframe — no allow-scripts */}
      {contentBody ? (
        <iframe
          sandbox="allow-same-origin"
          srcDoc={iframeContent}
          title="Newsletter content"
          className="w-full min-h-[300px] max-h-[60vh] rounded-lg border border-white/10 bg-transparent"
          style={{ colorScheme: "dark" }}
        />
      ) : (
        <p className="text-sm text-white/40 italic">Newsletter content unavailable</p>
      )}
    </div>
  );
}

Since the iframe has `sandbox="allow-same-origin"` (no scripts), `postMessage` won't work — manual height detection via `contentDocument.body.scrollHeight` is possible because same-origin iframes allow DOM access.

- **HTML body size limit** — Added 2MB max check in webhook handler. Prevents oversized payloads from hitting DB storage.
- **Type guard for approval metadata** — Replaced unsafe `as any` cast with proper `isNewsletterApproval()` type guard in TaskDetailPanel.
- **Date serialization** — Ensured `receivedAt` is serialized to ISO string in the pending route response to match `PendingNewsletterSummary` type.
